# 契約: アクセス統計（GET /links/{code}/stats）

> **段階**: ④ Plan の補助（`plan.md` §4 の詳細）
> **対応する設計**: `./../plan.md`  |  **対応する仕様**: `./../spec.md`
> **作成日**: 2026-05-31
>
> ※ テンプレートの動作例。`templates/contract-template.md` を雛形に書いた。

---

## 1. 概要

- **種別**: HTTP API
- **責務**: 指定された識別子の累計アクセス回数を返す。
- **対応する仕様**: US-3 / AC-3.1, AC-3.2

## 2. シグネチャ / エンドポイント

```
GET /links/{code}/stats
```

## 3. 入力

| 名前 | 型 | 必須 | 制約 | 説明 |
|------|----|----|------|------|
| code | string（パス） | ✓ | 7文字 Base62 | 短縮識別子 |

## 4. 出力（正常系）

| フィールド | 型 | 説明 |
|------------|----|------|
| code | string | 識別子 |
| hits | integer | 累計アクセス回数（未アクセスなら 0） |

- **成功の判定**: HTTP 200 OK

## 5. エラー（異常系）

| 条件 | 返し方 | 呼び出し側の対処 |
|------|--------|------------------|
| code が存在しない | 404 Not Found | 識別子を確認 |

## 6. 事前条件 / 事後条件

- **事前条件**: なし。
- **事後条件**: 状態を変えない（読み取り専用。`hits` は加算しない）。
- **冪等性**: 冪等（何度呼んでも同じ結果・副作用なし）。AC-3.2 のとおり未アクセスは 0。
